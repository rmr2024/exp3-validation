# Experiment 3 - Client-side Validation

This experiment demonstrates client-side validation for both the e-book and e-ticketing pages.

## Contents

- `ebook-login.html` — login/register form with validation.
- `ticket-booking.html` — booking form with travel details validation.
- `validation.js` — modern JavaScript validation logic.
- `styles.css` — basic styling for the sample pages.
